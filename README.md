# Iptables Manager
## Opening Port:

```
Command: open <port>
```


## Closing Port:
```
Command: Close <port>
```


## Remove Rule:
```
Command: remove <port>
```



## Features
```Command: set``` Set Open 22, 80, 443 And Close All Not Included And Close Ping.

```Command: unset``` Set Open 22, 80, 443 And Open All Not Included And Open Ping.
